# **App Name**: Adil Morocco

## Core Features:

- Identity and Authentication: Complete functional user system for registration, login, and secure password recovery with verified backend authentication.
- Interactive Media Feed: A dynamic scrolling feed of image and video posts allowing users to engage through real-time likes and nested comments.
- Maroccan Reels: An immersive vertical video playback system for browsing short-form content with high-speed delivery.
- AI Discovery Matcher Tool: A generative AI tool that analyzes user interaction patterns and uses reasoning to suggest relevant content and new accounts to follow.
- Cloud Asset Profiles: Fully customizable user profiles featuring personal biographies and cloud-integrated storage for avatars and multimedia galleries.
- Live Signal Center: A notification hub that manages and delivers instant alerts for new followers, comments, and engagement on user content.
- Omni-Search Bar: Real-time query tool for instantly finding other users and exploring trending hashtags across the platform.

## Style Guidelines:

- Primary color is Saffron Sun (#E36B3B), a rich and vibrant orange inspired by Moroccan markets.
- Background is Atlas Night (#1D1A19), a deep earthy dark brown that provides a high-end feel and enhances media vibrancy.
- Accent color is Hibiscus Rouge (#BD1B43), used for interactions and alerts, offering a sharp contrast to the primary orange.
- A font pairing of 'Playfair' for high-contrast headlines and 'PT Sans' for body text is used to balance elegance with readability.
- Icons are thin-stroked and minimalist, using metallic gold tints for secondary states.
- An edge-to-edge media-centric layout that emphasizes full-screen video content and high-quality photography.
- Liquid smooth transitions between reels and home feed to simulate the fluid nature of desert dunes.
